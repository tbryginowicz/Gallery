# Photo Gallery

Simple web application for managing photo galleries with role-based authorization.

## Tech Stack

- Java 17
- Spring Boot
- PostgreSQL
- Thymeleaf
- Docker

## Quick Start

```bash
docker-compose up --build
```

Application available at: http://localhost:8080

## Default Credentials

- **Admin**: `admin` / `admin`

## Features

- Login with roles (Admin/User)
- Admin panel (user, gallery, and photo management)
- Photo viewer for individual photos
- Access control (users see only their galleries)
